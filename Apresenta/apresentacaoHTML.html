Funcionamento e detalhes do sistema

Apresentação

O sistema inicia na tela de apresentação, aonde constam dois botões, um no qual serve para realizar o cadastro de um novo usuário e outro botão aonde irá conseguir visualizar os cadastros realizados. Ainda na tela de apresentação tens um link de acessos ao qual podes acessar o código fonte deste site.

Cadastro

Na tela de cadastros você terá um botão para voltar ao início, e um botão para acessar a tela de visualização de cadastros. Ainda no decorrer desta página temos o formulário cadastros para um novo usuário, que após preenchido deve ser selecionado o botão cadastrar para salvar os novos dados de um usuário. Ainda na tela de cadastro tens um link de acessos ao qual podes acessar o código fonte deste site.

Consulta

Na tela de consulta você terá um botão para voltar a página inícial e um botão para voltar a tela de cadastro de usuário. No corpo do site quando possuir cadastros de usuários ficará visivel, quando estiver em branco significa que não possui nenhum cadastro de usuário. Ao lado de cada linha com o cadastro do usuário tem um botão com a opção de deletar o usuário, o qual o cadastro do mesmo será excluido permanentemente. Ainda na tela de consulta tens um link de acessos ao qual podes acessar o código fonte deste site.
